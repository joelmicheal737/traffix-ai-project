# Traffic Datasets

This directory contains sample traffic data for Coimbatore locations.

## Files

- `coimbatore_traffic_sample.csv` - Sample traffic data with timestamps, locations, vehicle counts, speeds, and congestion levels

## Data Format

### CSV Columns:
- `timestamp` - Date and time of measurement
- `location` - Traffic monitoring location in Coimbatore
- `vehicle_count` - Number of vehicles detected
- `avg_speed` - Average speed in km/h
- `congestion_level` - Traffic congestion (low, medium, high, very_high)
- `weather` - Weather conditions
- `day_of_week` - Day of the week

## Key Locations

- **Gandhipuram** - Major commercial hub
- **RS Puram** - Residential and commercial area
- **Peelamedu** - IT corridor and residential
- **Saibaba Colony** - Residential area near PSG institutions

## Usage

Upload these CSV files through the web interface to populate the system with sample data for testing predictions and visualizations.
