# Demo Videos

This directory is intended for demo videos used in vehicle detection and counting.

## Supported Formats
- MP4, AVI, MOV
- Recommended resolution: 720p or higher
- Duration: 30 seconds to 5 minutes for demo purposes

## Usage

1. Place traffic videos in this directory
2. Use the Upload page to submit videos for YOLOv8 analysis
3. View detection results with vehicle counts and bounding boxes

## Sample Video Sources

For testing, you can use:
- Traffic camera footage
- Dashcam recordings
- Drone footage of intersections
- Mobile phone recordings of traffic

**Note**: Ensure you have proper permissions for any video content used.
